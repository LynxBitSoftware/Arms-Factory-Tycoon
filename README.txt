--*IDEA PROJECTE TYCOON*--

->RESUM: 

    El projecte "ww2-tycoon" será un videojoc 2D per Android, del tipus tycoon (manage resourcer) fent servir . 
    El joc tractaría de controlar i millorar la producció d'una fabrica de caràcter militar en plena WW2, 
    aquesta frabrica estaría situada en USA (primera fabrica) i allà començaria el joc, arribant a una nova 
    indústria buida i fer-la creixer poc a poc. 

->ELEMENTS CONTINGUTS AL JOC:
    
    -Diners($) // Recurs bàsic

    -Rubis // Recurs per millores -> $$

    -Bonificacions de temps i multiplicador d'income // Per X rubis, inclos en el menu d'upgrade

    -Personatges traballant (tenen nivell) // Ells "crean" els items cada X temps.

    -Personatges traballant en el transport (tenen nivell) // Ells "carregan" els camions de items cada X temps.

    -Braç que mou els items del treballador a una cinta transportadora. // personatge especial que s'encarrega de 
    moure els items d'una part de la cinta a l'altre OR augmentar tamany cinta transportadora 

    -Cinta transportadora de materials.

    -Filtre d'items // els porta a la zona de transport i funciona com multiplicador.

    -transport de items: vehicle militar de suministres. // son carregats per treballadors i s'envan 
    amb materials i tornen amb diners, que son ingresats a la conta del jugador

    -Boto d'upgrade // conté millores tant de el valor del item que estem fabricant, com la velocitat dels 
    treballadors, la velocitat/capacitat del braç transportador, millores del transport de suministres,
    millores del filtre(multiplicador),prestigi(multiplicador segons avanç) etc.

    -Boto carrito // conté tant el menu de pago amb €, comprar rubis,comprar Diners,  bonificacions de temps i
     multiplicadors, com l'eliminació d'anuncis in-game.

    -Porta // es desbloqueja per  Diners i et permet avançar a la seguent part de la fabrica. Permetent contractar
     més treballadors per fabricar nou material, més valios generalment.

    -Sistema de nivells // milloren l'efectivitat i productivitat d'un treballador, filtre, transport o braç.

    -Sistema de recompenses de rubis per veure anuncis i aconseguir "logros".
